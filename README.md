Gura Assistant

Gura Assistant is an open-source AI assistant built with Python and Flask. It combines conversational AI with long-term memory, secure authentication, and personal finance tools to create a practical assistant that can be self-hosted and extended by the community.

✨ Features

- 🤖 AI-powered conversations
- 🧠 Persistent long-term memory
- 🔒 Secure login authentication
- 💰 Fixed Deposit (FD) management
- ☁️ Cloud deployment on Render
- 🗄️ PostgreSQL database (Neon)
- 📱 Cross-device access (desktop & mobile)
- 🔄 Persistent chat history

🛠️ Tech Stack

- Python
- Flask
- PostgreSQL (Neon)
- Groq API
- Render
- HTML, CSS & JavaScript

Installation

Clone the repository:

git clone https://github.com/gurashish3628/assistant-gura.git

Install dependencies:

pip install -r requirements.txt

Create a ".env" file with your required API keys and secrets.

Run the application:

python app.py
